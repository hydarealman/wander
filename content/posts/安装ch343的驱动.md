# 安装ch343的驱动
解压并进入驱动目录

unzip ch343ser_linux-main.zip
cd ch343ser_linux-main/driver

安装编译依赖(如果未安装)
驱动需要内核头文件和编译工具
image.png


编译驱动模块
在driver目录下执行

sudo make install


加载驱动并测试
先卸载可能已经自动加载的老驱动

sudo modprobe -r ch343

加载编译的模块 

sudo insmod ch343.ko
