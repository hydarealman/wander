# 安装ch343的驱动

解压并进入驱动目录

```C++
unzip ch343ser_linux-main.zip
cd ch343ser_linux-main/driver
```



安装编译依赖\(如果未安装\)

驱动需要内核头文件和编译工具

![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=MDNiOGE5ODJmNDA4ZmUxZGNlN2YwYWZkYTkyZDI4NzhfYzE5MzliODI0MmUzMmUyYzkwNmQ2YTUyNDBhNzhiYzJfSUQ6NzYzNzgwNjYyODkzMTAxMzU2NF8xNzgwNzMzNjM3OjE3ODA4MjAwMzdfVjM)



编译驱动模块

在driver目录下执行

```C++
sudo make install
```





加载驱动并测试

先卸载可能已经自动加载的老驱动

```C++
sudo modprobe -r ch343
```



加载编译的模块 

```C++
sudo insmod ch343.ko
```
